# SecondV4 - لعبة Pixel Adventure

لعبة مغامرات ثنائية الأبعاد مبنية باستخدام **Pygame**، حيث يجب على اللاعب جمع العملات والمفتاح والوصول إلى الباب للفوز، مع تجنب الوحوش والحفر والرياح الخطرة.

## متطلبات التشغيل

- Python 3.x
- Pygame (`pip install pygame`)

## طريقة اللعب

- **الاتجاهات**: استخدام أسهم لوحة المفاتيح (← → ↑ ↓) لتحريك الشخصية
- **الهدف**: جمع المفتاح → فتح الباب → الفوز باللعبة
- **تجنب**: الوحش، رياح الوحش، الحفر، رياح الحفر

## العناصر

| العنصر | الوصف | الحجم |
|--------|-------|-------|
| **Player** | شخصية اللاعب، تتحرك بشكل شبكي (60×55 بكسل) | 50×50 |
| **Monster** | وحش يتحرك بشكل عشوائي عند كل ضغطة مفتاح | 50×50 |
| **MonsterBadWind** | رياح سامة تتبع الوحش (2 لكل وحش) | 50×50 |
| **Coin** | عملات قابلة للجمع (10 عملات) | 30×30 |
| **Key** | مفتاح لفتح الباب | 30×30 |
| **Door** | الباب النهائي للفوز | 50×50 |
| **Hole** | حفر تؤدي إلى الموت الفوري (3 حفر) | 50×50 |
| **HoleWind** | رياح خطرة حول كل حفرة (2 لكل حفرة) | 50×50 |

## آلية اللعب

1. اللاعب يتحرك على شبكة ثابتة (خطوات 60 بكسل أفقياً، 55 بكسل عمودياً)
2. كل ضغطة مفتاح تسبب حركة اللاعب + حركة الوحش عشوائياً
3. جمع العملات يزيد الرصيد (يُطبع في الكونسول)
4. جمع المفتاح يسمح بفتح الباب
5. لمس أي عنصر خطر (وحش، رياح، حفرة) يؤدي إلى Game Over
6. فتح الباب يؤدي到 شاشة الفوز
